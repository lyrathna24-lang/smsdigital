# ប្រព័ន្ធផ្ទេរទិន្នន័យចំណាត់ថ្នាក់ និងកិត្តិយស

## ពិពណ៌នា
ប្រព័ន្ធនេះអនុញ្ញាតឱ្យគ្រូបង្រៀនអាចផ្ទេរទិន្នន័យពិន្ទុប្រចាំខែទៅកាន់តារាងចំណាត់ថ្នាក់ និងតារាងកិត្តិយសបានដោយស្វ័យប្រវត្តិ។

## លក្ខណៈពិសេស
- ✅ ផ្ទេរទិន្នន័យពិន្ទុប្រចាំខែទៅតារាងចំណាត់ថ្នាក់
- ✅ ទាញសិស្សចំណាត់ថ្នាក់លេខ 1-5 ទៅតារាងកិត្តិយស
- ✅ រក្សាទុកទិន្នន័យក្នុង localStorage និងមូលដ្ឋានទិន្នន័យ
- ✅ គណនាចំណាត់ថ្នាក់ និងនិទ្ទេសដោយស្វ័យប្រវត្តិ
- ✅ បង្ហាញរបាយការណ៍ចំណាត់ថ្នាក់ដែលអាចបោះពុម្ពបាន

## របៀបប្រើប្រាស់

### ១. បញ្ចូលពិន្ទុប្រចាំខែ
1. ចុចលើឈ្មោះខែ (ឧ: "ខែធ្នូ") ដើម្បីបើកតារាងពិន្ទុប្រចាំខែ
2. បញ្ចូលពិន្ទុសិស្សគ្រប់មុខវិជ្ជា
3. ប្រព័ន្ធនឹងគណនាពិន្ទុសរុប និងមធ្យមភាគដោយស្វ័យប្រវត្តិ

### ២. ផ្ទេរទៅតារាងចំណាត់ថ្នាក់ និងកិត្តិយស
1. ចុចលើប៊ូតុង "ចំណាត់ថ្នាក់" នៅខាងក្រោមឈ្មោះខែ
2. ប្រព័ន្ធនឹង:
   - គណនាចំណាត់ថ្នាក់សិស្សតាមលំដាប់ពិន្ទុ
   - ផ្ទេរទាំងអស់សិស្សទៅតារាងចំណាត់ថ្នាក់
   - ទាញសិស្សចំណាត់ថ្នាក់លេខ 1-5 ទៅតារាងកិត្តិយស
   - រក្សាទុកទិន្នន័យក្នុងមូលដ្ឋានទិន្នន័យ
3. ទំព័រចំណាត់ថ្នាក់នឹងបើកដោយស្វ័យប្រវត្តិដើម្បីបង្ហាញលទ្ធផល

### ៣. មើលរបាយការណ៍
- **តារាងចំណាត់ថ្នាក់**: បង្ហាញសិស្សទាំងអស់តាមលំដាប់ចំណាត់ថ្នាក់
- **តារាងកិត្តិយស**: បង្ហាញតែសិស្សចំណាត់ថ្នាក់លេខ 1-5
- អាចបោះពុម្ពរបាយការណ៍បាន

## រចនាសម្ព័ន្ធមូលដ្ឋានទិន្នន័យ

### តារាងចំណាត់ថ្នាក់ប្រចាំខែ (monthly_rankings)
- `student_id`: លេខសម្គាល់សិស្ស
- `month_name`: ឈ្មោះខែ
- `semester`: ឆមាស (1 ឬ 2)
- `total_score`: ពិន្ទុសរុប
- `average`: មធ្យមភាគ
- `rank`: ចំណាត់ថ្នាក់
- `grade`: និទ្ទេស

### តារាងកិត្តិយស (honor_rolls)
- រចនាសម្ព័ន្ធដូចគ្នានឹងតារាងចំណាត់ថ្នាក់
- ផ្ទុកតែសិស្សចំណាត់ថ្នាក់លេខ 1-5

## ការកំណត់ថ្នាក់ដែលបានគណនា
- **ល្អណាស់**: មធ្យមភាគ ≥ 9.00
- **ល្អ**: មធ្យមភាគ ≥ 8.00
- **ល្អបង្គួរ**: មធ្យមភាគ ≥ 6.50
- **មធ្យម**: មធ្យមភាគ ≥ 5.00
- **ខ្សោយ**: មធ្យមភាគ < 5.00

## ឯកសារ API
- `api_ranking.php`: API សម្រាប់ការផ្ទេរទិន្នន័យទៅមូលដ្ឋានទិន្នន័យ
- `class.Database.php`: ថ្នាក់ភ្ជាប់មូលដ្ឋានទិន្នន័យ

## កំហុសទូទៅ
- ប្រសិនបើមិនមានទិន្នន័យពិន្ទុប្រចាំខែ នឹងមានបង្ហាញសារជូនដំណឹង
- ប្រសិនបើមិនអាចភ្ជាប់ទៅមូលដ្ឋានទិន្នន័យ ទិន្នន័យនឹងត្រូវបានរក្សាទុកក្នុង localStorage តែប៉ុណ្ណោះ

## ការអានបន្ថែម
ប្រព័ន្ធនេះត្រូវបានអភិវឌ្ឍដើម្បីជួយដល់គ្រូបង្រៀនក្នុងការគ្រប់គ្រងពិន្ទុសិស្ស និងបង្កើតតារាងចំណាត់ថ្នាក់ដោយងាយស្រួល។
